# diplome_draft

# [RU]

Результаты моделирования величин сдвигов ЭГХ и ЭИФ как графики для дипломной работы с использованием MATLAB.

**ВАЖНО:** на графиках пространственных сдвигов на самом деле размерность в метрах; то есть нормировка на $ \lambda_0 $ отсутствовала, отчего название ординаты некорректно.
